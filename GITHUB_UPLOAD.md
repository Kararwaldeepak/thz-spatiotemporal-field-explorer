# Put this repository on GitHub

## Browser method

1. Sign in to GitHub.
2. Select **New repository**.
3. Choose a repository name, for example:
   `spatiotemporal-thz-electric-field`
4. Create an empty repository.
5. Upload all files from this folder.
6. Commit the files.

## Git command-line method

```bash
git init
git add .
git commit -m "Initial THz spatiotemporal analysis repository"
git branch -M main
git remote add origin YOUR_GITHUB_REPOSITORY_URL
git push -u origin main
```
